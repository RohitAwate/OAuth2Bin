OAuth 2.0 Bin

OAuth 2.0 Test Server

OAuth 2.0 Bin is a free and open-source OAuth 2.0 test server which adheres to RFC 6749.
It uses a set of static parameters that you must use while making your request. These are detailed below.

  • Authorization Code
  • Implicit
  • Resource Owner Password Credentials
  • Client Credentials
Parameters

{{.AuthCodeCnfg.AuthURL}}

{{.AuthCodeCnfg.TokenURL}}

{{.AuthCodeCnfg.ClientID}}

{{.AuthCodeCnfg.ClientSecret}}
Authorization Grant Request

{{.AuthCodeCnfg.AuthURL}}?response_type=code&client_id={{.AuthCodeCnfg.ClientID}}&redirect_uri=https://my-redirect-uri.com&scope=hello-world

response_type=code required

Indicates that the application is requesting for an authorization grant token.

redirect_uri=... required

Redirects the user-agent to this address when the authorization is complete.

scope=... optional

You can show your creativity here as the server ignores this.
